# Teeteestock 行動端交易終端設計規格 (Mobile Trading Terminal - Option A)

## 1. 介面架構 (Architecture)

採用「單頁面多模式」架構，透過底部導航列 (Bottom Navigation) 切換顯示內容，減少頁面跳轉的破碎感。

### 1.1 佈局組成 (Composition)
- **Top Ticker (固定頂部)**: 即時公告跑馬燈。
- **Global Header (固定頂部)**: 顯示指數總和、漲跌幅、成交量與可用餘額。
- **Main Viewport (中央滾動區)**: 根據導航切換內容。
- **Bottom Navigation (固定底部)**: 核心功能切換鈕（自選、交易、資產）。

---

## 2. 功能模組設計

### 2.1 全域資訊列 (Top Section)
- **跑馬燈**: 背景 `#181a20`，文字 `#ff69b4`，顯示系統公告與重大事件。
- **資訊條**: 背景 `#181a20`，顯示 `TEE 指數`、`漲跌點數/百分比`、`總成交量`。

### 2.2 核心頁面 (Viewport Modes)
- **[自選] 模式**:
  - 股票清單：按筆劃順序，顯示代號、名稱、即時價、漲跌幅。
  - 漲跌色盤：紅漲綠跌。
  - 點擊跳轉至 [交易] 模式。
- **[交易] 模式 (核心畫面)**:
  - **頂部**: 當前選中股票名稱、現價、漲跌。
  - **中部**: K 線圖 (CandlestickChart)。
  - **下部**: 標籤頁切換（五檔報價、成交明細、社群動態）。
  - **底部**: 買進/賣出快捷按鈕（懸浮或固定於標籤下方）。
- **[資產] 模式**:
  - 顯示總資產估值、帳戶餘額、目前持股盈餘分析。

### 2.3 社群整合 (X/Live)
- 在 [交易] 模式的標籤頁中，顯示與該組合相關的 X 推文縮影或直播連結。

---

## 3. 視覺風格 (Aesthetics)
- **主題**: 深色 (Dark Mode)。
- **背景**: `#0b0e11`。
- **卡片/背景**: `#181a20`。
- **邊框**: `#2b2f36`。
- **強調色**:
  - 亮紅 (漲): `#ff3b3b`。
  - 亮綠 (跌): `#00ffa3`。
  - 品牌粉: `#ff69b4`。

---

## 4. 技術實作建議
- **Responsive Design**: 使用 Tailwind CSS 的 `flex-col` 與 `h-screen` 確保介面填滿行動端螢幕。
- **State Management**: 利用現有的 `TeeContext` 管理全局股票數據與切換狀態。
- **Components**:
  - 重構 `CandlestickChart` 以適應較小的容器寬度。
  - 新增 `BottomNav` 組件。
  - 新增 `TickerTape` 組件。

---

## 5. 驗證標準
1. 頂部資訊條在所有頁面保持一致。
2. 點擊自選清單後能流暢切換至該股票的交易視圖。
3. 底部導航在手機螢幕上佔據合理空間且易於點擊。
