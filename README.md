Teeteestock-web
VTuber 貼貼動態身價交易所 | バーチャルユーチューバー（主にホロライブ）のてぇてぇを株にするプロジェクト  | The sentiment market for VTuber dynamics.
TeeteeStock 是一個將 VTuber 組合（CP）的互動感（てえてえ）股票化的平台。
透過大眾的「意見點數 (OPS)」，以及固定期間內互動狀況作為淨值，即時反映不同組合之間的相愛程度與社群情緒。
🚀 核心概念
⦁	標的物：以「組合」為單位（例如：MIKOMETO, OKAKORO）。
⦁	價值來源：直播互動、社群貼文、以及大眾對該組合「貼貼感」的認可。
⦁	目標：用數據視覺化呈現 VTuber 圈最動人的瞬間。
🛠 技術堆疊 (規劃中)
⦁	Frontend: Next.js + Tailwind CSS
⦁	Backend: Node.js / Python
⦁	Database: PostgreSQL + Redis
⦁	Real-time: WebSocket (Socket.io)
